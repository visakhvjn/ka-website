# Innovative & Strategic F.Z.E

Modern accounting, tax, and business advisory website for UAE businesses. Built with Next.js, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize content

Update brand details, contact info, and stats in [`data/site.ts`](data/site.ts). Service categories live in [`data/services.ts`](data/services.ts).
